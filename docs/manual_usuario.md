# Manual de Usuario

## Introducción
El chatbot institucional permite resolver dudas frecuentes relacionadas con:
- Horarios
- Becas
- Secretaría
- Contacto

## Cómo acceder
1. Entrar en la página web
2. Pulsar el icono del chatbot
3. Escribir una pregunta

## Ejemplos de preguntas
- ¿Cuál es el horario del centro?
- ¿Cómo solicito una beca?
- ¿Dónde está secretaría?

## Funcionalidades
- Respuestas automáticas
- Integración con Voiceflow
- Lectura de datos desde GitHub

## Posibles errores
- Si el chatbot no responde:
  - Revisar conexión a internet
  - Recargar la página
